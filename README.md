# 🌱 Agrinho 2026 - Agro Forte, Futuro Sustentável

## 📚 Sobre o Projeto

O **Agrinho 2026 - Agro Forte, Futuro Sustentável** é um projeto educacional desenvolvido para conscientizar estudantes, produtores rurais e a comunidade em geral sobre a importância da sustentabilidade no agronegócio.

A plataforma apresenta conteúdos informativos, recursos interativos e materiais educativos que demonstram como a produção agrícola pode caminhar lado a lado com a preservação ambiental, a segurança alimentar e o desenvolvimento sustentável.

---

## 🎯 Objetivo

Promover a educação ambiental e incentivar práticas agrícolas responsáveis, mostrando que é possível aumentar a produtividade no campo sem comprometer os recursos naturais das futuras gerações.

O projeto busca aproximar tecnologia, conhecimento e sustentabilidade, formando cidadãos mais conscientes sobre os desafios e oportunidades do agronegócio moderno.

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/gesianemoro-arch/Agrinho2026.git
```

Acesse a pasta do projeto:

```bash
cd Agrinho2026
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Caso ocorra algum problema de compatibilidade:

```bash
npm install --legacy-peer-deps
npm run dev
```

---

## 🚀 Como Executar

Após iniciar o projeto, acesse:
